NAME
 molior-archive-sources - create archive with all molior project source packages

SYNOPSIS
 molior-archive-sources PROJECT VERSION

DESCRIPTION
 molior-archive-sources downloads all latest source packages of a molior project and creates a tar archive.

OPTIONS

  PROJECT   Molior project name
  VERSION   Molior project version

EXIT STATUS
 The program returns 0 when the deployment creation was successful, 1 if something went wrong.

FILES
  ~/.moliorrc  User configuration file

EXAMPLE

    $ molior-archive-sources myproject 1.2
